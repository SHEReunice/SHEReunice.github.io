Git报错

does not have a commit checked out

查看了很多回答说是在base中自动创建了一个.git文件，把它删掉就好了，然后一直没有找到这个文件，这是一个隐藏文件，要打开文件管理中的查看→勾选显示隐藏的项目

勾选之后就可以找到隐藏的.git文件

把它删掉然后再次添加文件

这样就可以成功添加到本地仓库了，接下来再按正常操作传到github上

git push 不好使

解决：在push前面先pull
